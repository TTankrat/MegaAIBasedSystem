# MegaAIBasedSystem

Проект интеллектуальной системы автоматизированной разметки по данным рентгенографии ОГК, направленной на решение двух основных задач:
  - Сегментации областей легких и средостения с последующим расчетом кардиоторакального индекса (КТИ), визуализацей линий-ориентиров для контроля ротации пациента;
  - Выделения сегментарного строения легких на рентгенограммах в прямой и боковых проекциях.
## Содержание

В представленных файлах `.ipynb` отражен ход разработки системы.

Файл `UP_Nurakhunov.ipynb` включает в себя:
  - Задание аугментаций и формирование набороб данных для обучения и валидации моделей сегментации;
  - Разработку архитектур сверточных нейронных сетей для решения задачи сегментации;
  - Задание метрик оценки моделей, цикла обучения;
  - Обучение моделей;
  - Агрегация результатов обучения для проведения сравнительного анализа.

Файлы `P3_Nurakhunov_1.ipynb` и `P3_Nurakhunov_1_тест.ipynb` содержат похожую структуру (ошибка в ходе разработки). Версия с суфиксом `_тест` является более актуальной и включает в себя:
  - Обучение отобранных после проведения сравнительного анализа моделей с сохранением моделей и метрик;
  - Разработку алгоритма расчета КТИ по сегментационным маскам;
  - Тестирование алгоритма расчета КТИ на истинных масках набора данных для последующего сравнительного анализа ручных измерений КТИ эксперта;
  - Тестирование полного пайплайна обработки снимка (загрузка изображения -> предобработка -> подача на вход модели -> получение предсказанной сегментационной
  маски с выхода модели -> применение алгоритма расчета КТИ) на сформированном независимом наборе данных для проверки работы системы. Тестирование проводилось для трех лучших моделей;
  - Тестирование системы на ошибку масштабирования изображения (Предсказанная моделью маска имеет разрешние 256x256 пикселей. Для предоставления врачу визуализации,
  предсказанная маска увеличивается до размеров входного рентгена).

Файл `P3_Nurakhunov_2.ipynb` включает в себя:
  - Функции загрузки и проверки разработанных идеализированных шаблонов легких для последующего использования;
  - Функции проверки загрузки файла размеченных ключевых точек;
  - Тестирование аугментаций для использования;
  - Сохранение уменьшенных до размера входа будущих моделей (512x512 пикселей) версий исходных изображений высокого разрешения (около 3000x3000 пикселей);
  - Преобразование и сохранение ключевых точек под уменьшенные изображения;
  - Разработку алгоритма определения геометрических ключевых точек по известным маске сегментации (классы: легкие, фон) и анатомическим ключевым точкам;
  - Разработку алгоритма деформации шаблонов легких по совокупности анатомических и геометрических ключевых точек;
  - Разработку метода для аугментации малочисленного набора данных путем деформации рентгенов, и соответствующих им масок, с помощью сдвига ключевых точек
  (получение синтетических рентгенов с известными масками и располодением ключевых точек);
  - Разработку архитектур моделей для решения задачи детекции ключевых точек, а также мультизадачи по одновременным сегментации области легких и детекции ключевых точек;
  - Задание метрик оценки моделии, разработка функций цикла обучения на полном наборе данных и обучения с кросс-валидацией для более точной оценки модели на малом наборе данных.

    В настоящий момент идут работы по обучению детекторов ключевых точек.
